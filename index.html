Painting Game On Your 3DS!

Splatoon Legends, is a fan game of the first Splatoon game from Wii U. We want to make a port of it on 3DS platform, this will support Old 3DS and New 3DS as well.

Read More Join us on Discord

On every 3DS Platform

Our goal is to make Splatoon playable on 3DS and New 3DS

The team has 2 majors project : the first one is to Demake Splatoon using Unity, so we will recode anything from start as close as possible to the original game, then make a build for the 3DS platform. The second one, is a real Decompilation of the code of Splatoon, to be more precise, a decompilation of the RPX file to make the game playable on a lot of platforms.

Our work started in february 2023 when Delus1on_L released his first video about porting Splatoon on the 3DS. From there, a lot of peoples joined the project and now we are close to 50 collaborators.
