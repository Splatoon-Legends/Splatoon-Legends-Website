Welcome on the FAQ Page!

Here we will reply to every question you may ask.

How can I play the game? When will it release?

The game is very early in development, and so there is no release date yet. When it does release, you will need to obtain the files of Splatoon for the Wii U, such as using a tool like Dumpling.

Will there be a demo of the game? Can I test it?

When it's ready, a small demo will be released similar to the Global Testfire. This will include much less content than the final game, allowing players to try out weapons such as the shooter and the roller. Currently we are not accepting new testers, so please don't ask.

Is this a port or a new game?

Currently we are working on a demake in Unity, which will include all of the features from Splatoon 1 on the Wii U. We are also trying to disassemble the original Splatoon game to port it directly, but the remake is much more likely at the moment.

Will this just be a port of the game?

There are two versions: a soft demake, and a full port that will add a lot of features.

Will this be on old 3DS/2DS consoles, or only the new ones?

We are hoping to make the games run on all 3DS systems, not just the newer models.

How will the game control? How will aiming work?

The game will control similarly to its Wii U counterpart, including gyro functionality for aiming the camera. Users with New 3DS models or the Circle Pad Pro will be able to use traditional camera controls with the right analogue stick.

Will I be able to play the game online? Does it work with my Wii U friends?

The game will hopefully support online play using Pretendo, which will provide custom servers for the 3DS. Wii U crossplay is unlikely, as these versions use different code than the Wii U version.

Will the Battle Dojo be part of the game?

The Battle Dojo will be available to play locally using the 3DS Download Play app.

Do I need a copy of Splatoon to play it?

For the decompilation version (not the demake), the decrypted files are needed. How you obtain them is none of our business.

Will the game support 3D effects?

Yes it will, there’s not much of a cost to using it, plus you can also just turn it off if you feel like it.

Will the Splatoon game rom/cia you guys are making be free? Or will we have to pay in order to play?

Everything in this project is free.
